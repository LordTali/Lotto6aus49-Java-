Dies ist ein simpler Nachbau des Lotto-Klassikers "6 aus 49", der in Java programmiert wurde. 
Die Gewinne (falls es denn welche gibt) sind rein virtual, aber die Spannung hoffentlich echt!

ACHTUNG!
Obligatorische Warnung: Glücksspiel kann süchtig machen. 
Echtes Geld kann man hier zwar nicht verspielen, aber falls Sie zu solchen Abhängigkeiten neigen, wurden Sie hiermit gewarnt.

ACHTUNG!
Das Spiel wurde meinerseits ausgiebig auf Bugs getestet, aber da Gewinne hier wie auch im echten Lotto so selten fallen, konnte ich die Funktionalität nicht in vollem Maße überprüfen.
Eine Rückmeldung hierzu bzw. eine Meldung zu etwaigen Bugs ist daher sehr willkommen!
